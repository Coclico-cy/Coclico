Instalación

Requisitos del sistema

Coclico es una aplicación nativa de Windows basada en .NET 10 y WPF. Verifique que su sistema cumple con los requisitos mínimos:

  • Sistema operativo: Windows 10 21H2 o Windows 11 (64 bits)
  • RAM: Mínimo 4 GB — 8 GB recomendados (para el módulo de IA)
  • Disco: 500 MB para la aplicación + ~4 GB para el modelo GGUF
  • Permisos: Cuenta de administrador requerida (elevación UAC al inicio)
  • GPU (opcional): Tarjeta compatible con Vulkan para aceleración de IA
  • Runtime: .NET 10 Runtime incluido en el instalador

Pasos de instalación

  1. Descargue el último instalador Coclico-Setup-x64.exe desde la página de descarga.
  2. Haga doble clic en el instalador — Windows puede mostrar una advertencia de SmartScreen. Haga clic en "Más información" → "Ejecutar de todas formas".
  3. Acepte la elevación UAC. Coclico se instala en %ProgramFiles%\Coclico\.
  4. En el primer inicio, el LauncherWindow le pide elegir el modo de inicio: manual, con Windows o minimizado en la bandeja del sistema.
  5. Para usar el módulo de IA, coloque su modelo GGUF en %APPDATA%\Coclico\resource\model\ y nómbrelo IA-support-chat.gguf.
Coclico requiere derechos de administrador para acceder a las APIs del sistema Windows (WMI, registro, memoria del kernel). La elevación se solicita una sola vez al inicio mediante UAC.

Primer inicio

En el primer inicio, Coclico realiza un análisis inicial del sistema y muestra una pantalla de inicio (SplashWindow) mientras los servicios se inicializan. Esto dura generalmente entre 3 y 8 segundos.

El LauncherWindow aparece para configurar: modo de inicio, tema (Claro/Oscuro/Sistema), idioma y perfil de usuario.

Interfaz principal

Coclico usa una interfaz WPF-UI 4.x con una barra de navegación vertical a la izquierda. Cada icono corresponde a un módulo.

  • Navegación: Clic en un icono o use atajos de teclado
  • Bandeja: Cerrar la ventana la minimiza en la bandeja del sistema
  • Tema: Botón sol/luna en la esquina superior derecha
  • Notificaciones: Los toasts aparecen en la esquina inferior derecha

Los 8 módulos

Dashboard

Página de inicio de Coclico. Muestra en tiempo real: uso de CPU, RAM utilizada/disponible/total, espacio en disco por unidad, rendimiento de red, puntuación de salud del sistema y acciones rápidas.

Limpieza (Deep Clean)

Analiza y elimina archivos innecesarios del sistema — 10 categorías: archivos temporales de Windows, caché de navegadores, registros del sistema, papelera de reciclaje, archivos temp de usuario, caché de miniaturas, informes de errores, instaladores antiguos, caché DNS, prefetch.

RAM Cleaner

Libera memoria física mediante llamadas P/Invoke a kernel32, ntdll, advapi32, dnsapi e iphlpapi. Requiere el privilegio SeIncreaseQuotaPrivilege. Típicamente libera entre 200 y 800 MB.

App Scanner

Analiza las aplicaciones instaladas e identifica software no utilizado, grande, duplicado o al final de su vida útil.

Instalador Rápido

Despliegue de aplicaciones comunes con un clic: instalación silenciosa, instalaciones por lotes, verificación de integridad SHA256 y retroceso automático en caso de error.

Aplicaciones

Lista todas las aplicaciones instaladas desde 8 fuentes: Registro HKLM+HKCU, Steam, Epic Games, GOG Galaxy, Ubisoft Connect, EA App, Rockstar Games, Microsoft Store.

Flow Chains

Sistema de automatización visual con 28 tipos de nodos, 10 operadores de condición y 3 acciones de error. Ver paso a paso.

Ajustes

Configure todos los aspectos de Coclico: idioma, tema, color de acento (predeterminado: #6366F1 Índigo), categorías de limpieza, modo IA y Auditoría, y política de seguridad.

Flow Chains — Paso a paso

Paso 1 — Crear una nueva cadena

En el módulo Flow Chains, haga clic en "Nueva cadena" y asígnele un nombre y descripción.

Paso 2 — Agregar nodos

Haga clic en "+ Agregar nodo". Los 28 tipos de nodos (NodeType) cubren: procesos, registro, archivos, limpieza, memoria, red y flujo de control.

Paso 3 — Configurar condiciones

Cada nodo puede tener una condición de ejecución con 10 operadores: Equals, NotEquals, GreaterThan, LessThan, GreaterOrEqual, LessOrEqual, Contains, NotContains, IsEmpty, IsNotEmpty.

Paso 4 — Establecer OnErrorAction

  • ContinueNext (predeterminado) — registrar error y continuar con el siguiente nodo
  • StopChain — detener la ejecución inmediatamente
  • SkipToEnd — omitir nodos intermedios, ejecutar solo el último nodo

Paso 5 — Ejecutar y monitorear

Haga clic en "Ejecutar". El progreso en tiempo real muestra el estado de cada nodo. Programe la ejecución automática mediante el menú "Programar".

Coclico IA

Coclico incorpora un asistente de IA completamente local basado en LLamaSharp 0.26. Ningún dato sale de su máquina (Zero Cloud).

Dos contextos LLM aislados (dual-executor): uno para el chat del usuario, otro para el OptimizationEngineService en segundo plano (cada 30 segundos).

Atajos de teclado

  • Ctrl + 1–8 — Navegar a los módulos 1 al 8
  • Ctrl + Shift + C — Limpieza rápida desde cualquier lugar
  • Ctrl + Shift + R — Liberar RAM rápidamente
  • Ctrl + Shift + A — Abrir chat de IA
  • Escape — Cerrar panel/diálogo activo

Archivos y Datos

Todos los datos de Coclico residen en %APPDATA%\Coclico\: ajustes, flow chains, registros de auditoría, registros Serilog, instantáneas de reversión y el modelo de IA. No se envían datos a servidores externos.
