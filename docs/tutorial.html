Installation & Prérequis

Coclico est une application portable — pas d'installeur. Téléchargez, extrayez et lancez.

Configuration requise

Système d'exploitationWindows 10 22H2 (build 22621)+ · Windows 11
Architecturex64 uniquement (64-bit)
RAM minimum4 GB (8 GB recommandés avec IA)
Espace disque~300 MB (sans modèle IA) · ~4-6 GB avec IA
PrivilègesAdministrateur obligatoire (UAC)
GPU (optionnel pour IA)Vulkan ou CUDA compatible

Étapes d'installation

1

Télécharger l'archive

Rendez-vous sur la page GitHub Releases et téléchargez le fichier Coclico-v1.0.3-win-x64.zip.

2

Extraire l'archive

Faites un clic droit sur le ZIP → Extraire tout. Choisissez un dossier stable (ex. C:\Programs\Coclico). Évitez le Bureau ou Téléchargements.

3

Lancer en tant qu'Administrateur

Faites un clic droit sur Coclico.exe → Exécuter en tant qu'administrateur. La fenêtre UAC s'ouvre — cliquez Oui. L'élévation est obligatoire pour les opérations RAM et processus.

4

Fenêtre de démarrage (SplashScreen)

Au premier lancement, un écran de démarrage initialise les services. Durée : 2-5 secondes.

5

Fenêtre Lanceur (premier démarrage uniquement)

Une fenêtre de bienvenue vous propose de choisir votre langue, votre thème et de configurer le démarrage automatique.

⚠ Important : Si vous n'exécutez pas en tant qu'Administrateur, l'application affiche une invite UAC et se relance automatiquement avec les bons privilèges. C'est le comportement normal.

Activer le modèle IA (optionnel)

L'IA locale nécessite un fichier modèle GGUF séparé (non inclus dans l'archive).

1

Obtenir le modèle

Téléchargez un modèle GGUF compatible (ex. Llama 3.2 3B Instruct, Mistral 7B). Taille recommandée : 3-7B paramètres.

2

Placer le fichier

Copiez le fichier .gguf dans le dossier resource/model/ et renommez-le exactement : IA-support-chat.gguf.

3

Relancer Coclico

Le module IA détecte automatiquement le modèle au démarrage. Aucune configuration supplémentaire n'est requise.

💡 Pour activer l'accélération GPU, éditez le .csproj et remplacez LLamaSharp.Backend.Cpu par LLamaSharp.Backend.Vulkan, puis recompilez.
Démarrage

Première utilisation

Voici ce qui se passe lors du premier lancement et comment configurer rapidement Coclico selon vos préférences.

Séquence de démarrage

🌊

SplashWindow

Écran de chargement animé qui initialise tous les services (DI container, base de données programmes, services réseau, etc.).

🚀

LauncherWindow (premier lancement)

Fenêtre de bienvenue : choix de langue, thème, activation du démarrage automatique. N'apparaît qu'une seule fois.

🏠

MainWindow

La fenêtre principale s'ouvre avec le Tableau de Bord actif. Toute la navigation se fait via la barre latérale gauche.

Configuration rapide recommandée

  1. Allez dans Paramètres → choisissez votre langue et votre couleur d'accent
  2. Activez le démarrage automatique si vous souhaitez que Coclico se lance avec Windows
  3. Configurez votre profil utilisateur (nom + avatar) via l'icône de profil en bas de la sidebar
  4. Naviguez dans le module Applications pour vérifier que vos logiciels sont bien détectés
  5. Créez votre première Flow Chain avec une automatisation simple (voir section dédiée)
Navigation

L'interface générale

Coclico utilise une interface Fluent Design (WPF-UI 4.2) avec une barre de navigation latérale fixe et une zone de contenu centrale.

Structure de la fenêtre

Barre latérale gauche

Navigation entre les 8 modules + accès profil, tray, et paramètres. Peut être compacte ou large.

Zone de contenu

Affiche le module actif. Chaque module est un UserControl WPF indépendant chargé à la demande.

Barre de titre

Barre personnalisée WPF-UI avec boutons Réduire / Maximiser / Fermer. Intègre le titre de la page active.

Icône système (Tray)

Coclico se minimise dans la zone de notification. Clic gauche → ouvrir. Clic droit → menu rapide.

Modes de la sidebar

Deux modes disponibles dans les Paramètres :

  • Mode large — icônes + texte, idéal sur grands écrans
  • Mode compact — icônes seules, libère de l'espace sur petits écrans
💡 Vous pouvez réduire Coclico dans le tray en cliquant le bouton ×. L'application continue de tourner en arrière-plan (utile pour les raccourcis globaux et l'auto-clean RAM).
Module 01

Tableau de Bord

Le tableau de bord est la page d'accueil de Coclico. Il affiche en temps réel les métriques essentielles de votre système, actualisées toutes les 3 secondes.

Métriques surveillées

🖥️

CPU

Pourcentage d'utilisation du processeur, mesuré avec PerformanceCounter Windows. Affiché sous forme de jauge et de valeur numérique.

🧠

RAM utilisée / totale

Mémoire physique consommée en GB, avec le total disponible. Calculé via GlobalMemoryStatusEx (Win32 P/Invoke).

💾

Disque C:

Espace utilisé et total du disque système. Affiché en GB avec un pourcentage de remplissage.

⏱️

Uptime Windows

Temps écoulé depuis le dernier démarrage de Windows, en jours/heures/minutes.

⚙️

Processus actifs

Nombre total de processus en cours d'exécution sur le système.

📦

Apps installées

Nombre d'applications détectées depuis toutes les sources (registre + game launchers).

💡 Le polling de 3 secondes est intentionnellement léger pour ne pas impacter les performances. Les métriques sont lues via des appels P/Invoke natifs optimisés.

Module 03 — Clé

Flow Chains

Les Flow Chains sont le moteur d'automatisation de Coclico. Créez des séquences d'actions visuelles déclenchées par raccourci clavier, même quand Coclico est en arrière-plan.

Créer une Flow Chain

1

Cliquer "Nouvelle chaîne"

Bouton en haut à droite du module. Donnez un nom explicite à votre chaîne.

2

Ajouter un nœud Start

Tout Flow Chain commence par un nœud Start. Il est ajouté automatiquement à la création.

3

Glisser-déposer des nœuds

Faites glisser les nœuds depuis le panneau gauche vers le canvas. Connectez-les en tirant depuis le point de sortie vers l'entrée du nœud suivant.

4

Configurer chaque nœud

Cliquez sur un nœud pour ouvrir son panneau de propriétés. Configurez les paramètres, le délai de retry et le timeout.

5

Assigner un raccourci global

Dans la barre du haut, cliquez le champ raccourci et appuyez la combinaison souhaitée (Ctrl+Alt+K par exemple).

6

Sauvegarder et tester

Cliquez Sauvegarder puis le bouton ▶ Exécuter pour tester immédiatement. Les résultats de chaque nœud s'affichent en temps réel.

Les 28 types de nœuds

Contrôle du flux

Start End Condition Loop Delay Parallel

Applications & processus

OpenApp CloseApp KillProcess KillByMemory SetProcessPriority

Système & audio

SetVolume MuteAudio Screenshot EmptyRecycleBin CleanTemp RamClean Notification

Scripts & commandes

Fichiers

Conditions disponibles

Exemple de Flow Chain : Routine de travail

Voici un exemple concret — une chaîne qui prépare votre session de travail :

Start
  → OpenApp "Code.exe"          [Ouvre VS Code]
  → Delay 3s                    [Attendre le chargement]
  → OpenApp "Firefox.exe"       [Ouvre le navigateur]
  → SetVolume 60%               [Volume à 60%]
  → Condition : RamAbove 80%    [Si RAM > 80%]
    ├─ Oui → RamClean           [Nettoyer si nécessaire]
    └─ Non → Notification "Prêt !" [Notification sinon]
  → End
💡 Assignez le raccourci Ctrl+Alt+W à cette chaîne. Elle se déclenche même si Coclico est minimisé dans le tray, ou si vous êtes dans une autre application.

Comment nettoyer

1

Sélectionner les catégories

2

Lancer le nettoyage

Cliquez Nettoyer. Une barre de progression s'affiche. Durée : quelques secondes à une minute selon la quantité.

3

Consulter le rapport

Module 06

RAM Cleaner

Surveillance et optimisation avancée de la mémoire RAM via 18 opérations Windows natives. Le module le plus technique de Coclico.

Métriques surveillées en temps réel

RAM physiqueUtilisée / Totale en GB, avec %
Mémoire virtuelle (Commit)Espace d'adressage virtuel utilisé
PagefileUtilisation du fichier d'échange Windows
Historique 90 secondesGraphique temps réel de la consommation RAM
Top 8 processusLes processus les plus gourmands en mémoire
VRAM GPUMémoire dédiée et partagée du GPU

Les 3 profils de nettoyage

⚡

Profil Rapide (~2 opérations)

Working Sets + GC .NET forcé. Très rapide (< 1 seconde). Impact minimal sur le système. Idéal en usage quotidien.

🔧

Profil Normal (~8 opérations)

Profil Rapide + Standby Lists + Cache fichiers modifiés. Durée ~2-3 secondes. Recommandé pour le nettoyage régulier.

🧠

Profil Profond (18 opérations)

Toutes les opérations incluant Working Set noyau, Cache DNS, ARP, NetBIOS, SuperFetch, Compactage du tas. Durée ~5-10s. Utiliser avec modération.

Mode automatique

Activez le nettoyage automatique via deux modes :

  • Seuil RAM — Déclenche le nettoyage quand la RAM dépasse X% (ex. 85%)
  • Intervalle — Nettoie automatiquement toutes les N minutes

Le profil utilisé en mode auto est configurable indépendamment du profil manuel.

⚠ Le Profil Profond désactive temporairement SuperFetch et vide le cache de standby système. Cela peut ralentir légèrement les premiers accès aux fichiers après le nettoyage. À utiliser quand le système manque vraiment de RAM.

Intelligence Artificielle

Coclico AI

Assistant conversationnel entièrement local, propulsé par un modèle GGUF via LLamaSharp v0.26. Zéro donnée envoyée sur Internet.

Fonctionnalités

🔒

100% local et privé

Le modèle tourne entièrement sur votre machine. Aucune connexion réseau requise pour l'inférence.

📚

RAG intégré (Retrieval-Augmented Generation)

L'IA dispose d'une base de connaissances sur Coclico, construite à partir des docs Markdown dans resource/docs/. Elle peut répondre à des questions précises sur chaque module.

⚡

Actions directes

Dites "Ouvre le RAM Cleaner" ou "Lance un nettoyage" — l'IA interprète l'intention et exécute l'action via le FeatureExecutionEngine.

🌍

Multilingue

Répond dans la langue de l'utilisateur, quelle que soit la langue du système ou de Coclico.

Pro Tips

Conseils & Astuces

Optimiser les performances

  • Activez le mode auto du RAM Cleaner avec un seuil à 80% — Coclico nettoiera automatiquement sans intervention
  • Utilisez le Profil Rapide pour le nettoyage quotidien, réservez le Profil Profond aux situations critiques
  • Le Dashboard se met à jour toutes les 3s — laissez-le ouvert pour surveiller votre système pendant les tâches intensives

Tirer le meilleur des Flow Chains

  • Créez une chaîne "Fin de journée" : ferme les apps, vide la corbeille, nettoie la RAM, éteint le PC
  • Créez une chaîne "Jeu" : ferme les apps inutiles, RAM Clean profond, règle le volume, ouvre Steam
  • Utilisez les conditions RamAbove pour un nettoyage conditionnel intelligent
  • Le nœud Parallel permet d'exécuter plusieurs actions simultanément — idéal pour les ouvertures d'apps

Dépannage

  • Si l'IA ne répond pas : vérifiez que resource/model/IA-support-chat.gguf existe et est accessible
  • Si les raccourcis globaux ne fonctionnent pas : vérifiez qu'aucun autre logiciel n'utilise la même combinaison
  • Si des apps ne sont pas détectées : cliquez Rafraîchir dans le module Applications pour forcer une nouvelle détection
  • Consultez les logs dans %AppData%\Coclico\logs\ pour diagnostiquer tout problème
