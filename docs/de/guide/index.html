Installation

Systemanforderungen

Coclico ist eine native Windows-Anwendung auf Basis von .NET 10 und WPF. Stellen Sie sicher, dass Ihr System die Mindestanforderungen erfüllt:

  • Betriebssystem: Windows 10 21H2 oder Windows 11 (64-Bit)
  • RAM: Mindestens 4 GB — 8 GB empfohlen (für das KI-Modul)
  • Festplatte: 500 MB für die App + ca. 4 GB für das GGUF-Modell
  • Berechtigungen: Administratorkonto erforderlich (UAC-Erhöhung beim Start)
  • GPU (optional): Vulkan-kompatible Karte für KI-Beschleunigung
  • Runtime: .NET 10 Runtime ist im Installationsprogramm enthalten

Installationsschritte

  1. Laden Sie den neuesten Installer Coclico-Setup-x64.exe von der Download-Seite herunter.
  2. Doppelklicken Sie auf den Installer — Windows zeigt möglicherweise eine SmartScreen-Warnung an. Klicken Sie auf "Weitere Informationen" → "Trotzdem ausführen".
  3. Akzeptieren Sie die UAC-Erhöhung. Coclico wird in %ProgramFiles%\Coclico\ installiert.
  4. Beim ersten Start erscheint das LauncherWindow zur Konfiguration des Startmodus: manuell, mit Windows oder minimiert in der Taskleiste.
  5. Um das KI-Modul zu nutzen, legen Sie Ihr GGUF-Modell in %APPDATA%\Coclico\resource\model\ ab und benennen Sie es IA-support-chat.gguf.
Coclico benötigt Administratorrechte für den Zugriff auf Windows-System-APIs (WMI, Registry, Kernel-Speicher). Die Erhöhung wird einmalig beim Start über UAC angefordert.

Erster Start

Beim ersten Start führt Coclico eine erste Systemanalyse durch und zeigt einen Ladebildschirm (SplashWindow) an, während Dienste initialisiert werden. Dies dauert normalerweise 3 bis 8 Sekunden.

Das LauncherWindow erscheint dann zur Konfiguration von: Startmodus, Thema (Hell/Dunkel/System), Sprache und Benutzerprofil.

Hauptoberfläche

Coclico verwendet eine WPF-UI 4.x-Oberfläche mit einer vertikalen Navigationsleiste links. Jedes Symbol entspricht einem Modul.

  • Navigation: Klicken Sie auf ein Symbol oder verwenden Sie Tastenkürzel
  • Tray: Fenster schließen minimiert in die Taskleiste (Doppelklick zum Wiederherstellen)
  • Thema: Sonne/Mond-Schaltfläche oben rechts zum Umschalten
  • Benachrichtigungen: Toast-Nachrichten erscheinen unten rechts

Die 8 Module

Dashboard

Die Startseite von Coclico zeigt in Echtzeit: CPU-Auslastung, RAM-Nutzung, Festplattenplatz, Netzwerkdurchsatz, System-Gesundheitsbewertung und Schnellaktionen.

Bereinigung (Deep Clean)

Analysiert und entfernt unnötige Dateien aus dem System — 10 Kategorien: Windows-Temp, Browser-Cache, Systemprotokolle, Papierkorb, Benutzer-Temp, Miniaturansichten-Cache, Fehlerberichte, alte Installer, DNS-Cache, Prefetch.

RAM Cleaner

Gibt physischen Arbeitsspeicher frei durch P/Invoke-Aufrufe an kernel32, ntdll, advapi32, dnsapi und iphlpapi. Benötigt das Privileg SeIncreaseQuotaPrivilege. Typischerweise werden 200 bis 800 MB freigegeben.

App Scanner

Analysiert installierte Anwendungen und identifiziert ungenutzte, große, doppelte oder veraltete Software.

Schnell-Installer

Deployment gängiger Anwendungen mit einem Klick: stille Installation, Batch-Installationen, SHA256-Integritätsprüfung, automatisches Rollback bei Fehlern.

Anwendungen

Listet alle installierten Anwendungen aus 8 Quellen: Registry HKLM+HKCU, Steam, Epic Games, GOG Galaxy, Ubisoft Connect, EA App, Rockstar Games, Microsoft Store.

Flow Chains

Visuelles Automatisierungssystem mit 28 Knotentypen, 10 Bedingungsoperatoren und 3 Fehleraktionen. Siehe Schritt-für-Schritt-Anleitung.

Einstellungen

Konfigurieren Sie alle Aspekte von Coclico: Sprache, Thema, Akzentfarbe (Standard: #6366F1 Indigo), Bereinigungskategorien, KI & Audit-Modus und Sicherheitsrichtlinie.

Flow Chains — Schritt für Schritt

Schritt 1 — Neue Kette erstellen

Klicken Sie im Flow Chains-Modul auf "Neue Kette" und geben Sie einen Namen und eine Beschreibung ein.

Schritt 2 — Knoten hinzufügen

Klicken Sie auf "+ Knoten hinzufügen". Die 28 Knotentypen (NodeType) umfassen: Prozesse, Registry, Dateien, Bereinigung, Arbeitsspeicher, Netzwerk und Steuerungsfluss.

Schritt 3 — Bedingungen konfigurieren

Jeder Knoten kann eine Ausführungsbedingung mit 10 Operatoren haben: Equals, NotEquals, GreaterThan, LessThan, GreaterOrEqual, LessOrEqual, Contains, NotContains, IsEmpty, IsNotEmpty.

Schritt 4 — OnErrorAction festlegen

  • ContinueNext (Standard) — Fehler protokollieren, weiter mit dem nächsten Knoten
  • StopChain — Ausführung sofort beenden
  • SkipToEnd — alle Zwischenknoten überspringen, nur letzten Knoten ausführen

Schritt 5 — Ausführen und überwachen

Klicken Sie auf "Ausführen". Der Echtzeit-Fortschritt zeigt den Status jedes Knotens. Planen Sie die automatische Ausführung über das Menü "Planen".

Coclico KI

Coclico enthält einen vollständig lokalen KI-Assistenten auf Basis von LLamaSharp 0.26. Keine Daten verlassen Ihren Computer (Zero Cloud).

Zwei isolierte LLM-Kontexte (Dual-Executor): ein Kontext für den Benutzer-Chat, ein anderer für den OptimizationEngine-Hintergrundprozess (alle 30 Sekunden).

Tastenkürzel

  • Strg + 1–8 — Navigation zu den Modulen 1 bis 8
  • Strg + Umschalt + C — Schnellbereinigung
  • Strg + Umschalt + R — RAM schnell freigeben
  • Strg + Umschalt + A — KI-Chat öffnen
  • Escape — Aktives Panel/Dialog schließen

Dateien & Daten

Alle Coclico-Daten befinden sich in %APPDATA%\Coclico\: Einstellungen, Flow Chains, Audit-Protokolle, Serilog-Protokolle, Rollback-Snapshots und das KI-Modell. Es werden keine Daten an externe Server gesendet.
