Guide & Tutoriel

Tout ce qu'il faut pour installer, configurer et maîtriser Coclico — du premier lancement aux automatisations avancées.

📚 Documentation API Référence complète des interfaces, services et modèles ⚙️ Wiki Développeur Architecture interne, MVVM, DI et système autonome ❓ FAQ Questions fréquentes sur l'installation et l'utilisation

Installation

Configuration requise

Coclico est une application Windows native basée sur .NET 10 et WPF. Avant d'installer, vérifiez que votre système répond aux exigences minimales :

  • OS : Windows 10 21H2 ou Windows 11 (64 bits)
  • RAM : 4 Go minimum — 8 Go recommandé pour le module IA
  • Disque : 500 Mo pour l'application + espace pour le modèle GGUF (~4 Go)
  • Permissions : Compte administrateur requis (élévation UAC au démarrage)
  • GPU (optionnel) : Carte compatible Vulkan pour l'accélération du modèle IA
  • Runtime : .NET 10 Runtime inclus dans l'installeur

Étapes d'installation

  1. Téléchargez le dernier installeur Coclico-Setup-x64.exe depuis la page de téléchargement.
  2. Double-cliquez sur l'installeur — Windows peut afficher un avertissement SmartScreen. Cliquez sur « Informations complémentaires » → « Exécuter quand même ».
  3. Acceptez l'élévation UAC. L'installeur place Coclico dans %ProgramFiles%\Coclico\.
  4. Au premier lancement, la LauncherWindow vous demande de choisir le mode de démarrage : manuel, avec Windows, ou réduit dans la barre système.
  5. Pour utiliser le module IA, placez votre modèle GGUF dans %APPDATA%\Coclico\resource\model\ et nommez-le IA-support-chat.gguf.
Coclico nécessite des droits administrateur pour accéder aux API système (WMI, registre, mémoire noyau). L'élévation est demandée une seule fois au démarrage via UAC.

Premier lancement

Au premier lancement, Coclico effectue une analyse initiale du système et affiche un écran de démarrage (SplashWindow) pendant l'initialisation des services. Cela prend généralement 3 à 8 secondes.

La LauncherWindow apparaît ensuite pour configurer :

  • Mode de lancement : Manuel / Au démarrage de Windows / Réduit dans le tray
  • Thème : Clair, Sombre, ou Système (suit les préférences Windows)
  • Langue : Français, English, Deutsch, Español
  • Profil utilisateur : Nom et avatar (optionnel)

Interface principale

Coclico utilise une interface WPF-UI 4.x avec une barre de navigation verticale sur la gauche. Chaque icône correspond à un module. La barre de titre est personnalisée avec des boutons de contrôle de fenêtre intégrés.

  • Navigation : Cliquez sur une icône ou utilisez les raccourcis clavier
  • Tray : Fermer la fenêtre la réduit dans la barre système (double-clic pour rouvrir)
  • Thème : Bouton soleil/lune en haut à droite pour basculer clair/sombre
  • Notifications : Les toasts apparaissent en bas à droite pour les actions importantes

Les 8 modules

Tableau de bord

Le Tableau de bord est la page d'accueil de Coclico. Il affiche en temps réel :

  • CPU : Utilisation instantanée et moyenne sur 60 secondes
  • RAM : Mémoire utilisée / disponible / totale avec graphique historique
  • Disque : Espace utilisé par disque avec alertes si <10% libre
  • Réseau : Débit entrant/sortant et adresse IP locale
  • Score santé système : Indicateur global calculé par StartupHealthService
  • Actions rapides : Boutons nettoyage rapide, libération RAM, scan direct

Nettoyage profond

Le module Nettoyage profond analyse et supprime les fichiers inutiles de votre système en 10 catégories :

  1. Fichiers temporaires Windows — %TEMP% et C:\Windows\Temp
  2. Cache navigateurs — Chrome, Edge, Firefox, Opera (profils auto-détectés)
  3. Logs système — Journaux d'événements Windows, logs IIS, logs de mise à jour
  4. Corbeille — Vide la corbeille sur tous les disques
  5. Temp utilisateur — AppData\Local\Temp par utilisateur
  6. Cache miniatures — thumbs.db et bases de données d'icônes Windows
  7. Rapports d'erreurs Windows — Dossier WER
  8. Anciens installeurs — .msi/.exe en attente dans le cache Windows Installer
  9. Cache DNS — Vidage via DnsFlushResolverCache (iphlpapi.dll)
  10. Prefetch — Fichiers prefetch Windows (C:\Windows\Prefetch)

RAM Cleaner

Le RAM Cleaner libère la mémoire physique en forçant Windows à réduire les working sets des processus et vider les listes standby. Il utilise directement des API Win32 via P/Invoke :

  • kernel32.dll — SetProcessWorkingSetSize, GetCurrentProcess
  • ntdll.dll — NtSetSystemInformation (SystemMemoryListInformation)
  • advapi32.dll — AdjustTokenPrivileges, LookupPrivilegeValue
  • dnsapi.dll — DnsFlushResolverCache
  • iphlpapi.dll — FlushIpNetTable

Typiquement, 200 à 800 Mo sont libérés selon l'état du système.

App Scanner

L'App Scanner analyse vos applications installées et identifie les logiciels inutilisés, volumineux, en double ou en fin de vie. Les résultats s'affichent dans une liste triable avec filtres.

Installeur rapide

Déployez des applications courantes en un clic via InstallerService. Supporte l'installation silencieuse, les installations par lot, la vérification d'intégrité SHA256 et le rollback automatique en cas d'échec.

Applications

Liste toutes les applications installées depuis 8 sources : Registre HKLM+HKCU, Steam, Epic Games, GOG Galaxy, Ubisoft Connect, EA App, Rockstar Games, Microsoft Store.

Flow Chains

Système d'automatisation visuel avec 28 types de nœuds, 10 opérateurs de condition et 3 actions sur erreur. Voir Flow Chains pas à pas pour le guide complet.

Paramètres

Configurez tous les aspects de Coclico : langue, thème, couleur d'accentuation (par défaut #6366F1 Indigo), catégories de nettoyage, mode IA & Audit, politique de sécurité et notifications.

Flow Chains — Pas à pas

Étape 1 — Créer une nouvelle chaîne

Dans le module Flow Chains, cliquez sur « Nouvelle chaîne ». Donnez-lui un nom et une description. Chaque chaîne contient une liste ordonnée de nœuds (PipelineNode).

Étape 2 — Ajouter des nœuds

Cliquez sur « + Ajouter un nœud » pour ouvrir le sélecteur de type. Les 28 types de nœuds disponibles (NodeType enum) couvrent la gestion des processus, le registre, les fichiers, le nettoyage, la mémoire, le réseau et le contrôle de flux.

Étape 3 — Configurer les conditions

Chaque nœud peut avoir une condition d'exécution avec 10 opérateurs : Equals, NotEquals, GreaterThan, LessThan, GreaterOrEqual, LessOrEqual, Contains, NotContains, IsEmpty, IsNotEmpty.

Étape 4 — Définir OnErrorAction

  • ContinueNext (par défaut) — log de l'erreur et continuation vers le nœud suivant
  • StopChain — arrêt immédiat de l'exécution de la chaîne
  • SkipToEnd — saut de tous les nœuds intermédiaires, exécution uniquement du dernier nœud

Étape 5 — Exécuter et surveiller

Cliquez sur « Exécuter ». La progression en temps réel affiche l'état de chaque nœud (En attente / En cours / Succès / Erreur). Planifiez l'exécution automatique via le menu « Planifier ».

Coclico IA

Coclico intègre un assistant IA entièrement local basé sur LLamaSharp 0.26. Aucune donnée ne quitte votre machine (Zero Cloud).

Deux contextes LLM isolés (dual-executor) garantissent que les tâches d'optimisation en arrière-plan ne ralentissent jamais votre chat interactif :

  • Contexte chat (_chatCtx) — dédié au chat utilisateur dans AiChatView
  • Contexte moteur (_engineCtx) — dédié aux tâches de fond de OptimizationEngineService (toutes les 30s)

Raccourcis clavier

  • Ctrl + 1–8 — Naviguer vers les modules 1 à 8
  • Ctrl + Maj + C — Nettoyage rapide depuis n'importe où
  • Ctrl + Maj + R — Libération RAM rapide
  • Ctrl + Maj + A — Ouvrir le chat IA
  • Échap — Fermer le panneau/dialogue actif

Fichiers & Données

Toutes les données Coclico se trouvent dans %APPDATA%\Coclico\ : paramètres, flow chains, logs d'audit, logs Serilog, snapshots de rollback et le modèle IA. Aucune donnée n'est envoyée à des serveurs externes.
