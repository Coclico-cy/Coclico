Coclico v1.0.3

Gestionnaire système Windows complet développé en .NET 10 / WPF-UI 4.2 / C# 13. Coclico centralise 8 modules actifs de surveillance, automatisation, nettoyage et intelligence artificielle dans une interface unifiée avec un thème sombre ultra-personnalisable.

Installation

Téléchargez la dernière release depuis la page GitHub, puis lancez Coclico.exe en tant qu'administrateur. Aucun installeur requis — format portable.

  1. Téléchargez le fichier Coclico.exe depuis la page de release
  2. Placez-le dans le dossier de votre choix (ex : C:\Apps\Coclico\)
  3. Installez le .NET 10 Desktop Runtime si nécessaire
  4. Lancez Coclico.exe en tant qu'administrateur
  5. Au premier lancement, l'application se configure automatiquement
Données utilisateur : Toutes les données sont stockées dans %AppData%\Coclico\. La désinstallation ne supprime pas vos données.
Modèle IA optionnel : Placez un fichier .gguf dans resource/model/IA-support-chat.gguf (à côté de Coclico.exe) pour activer Coclico AI. Le modèle n'est pas inclus dans la release.

Tableau de Bord

Page d'accueil de Coclico. Affiche en temps réel toutes les 3 secondes :

  • CPU % — utilisation processeur instantanée
  • RAM — utilisée / totale en GB
  • Disque C:\ — espace utilisé / total
  • Uptime — temps depuis le dernier démarrage Windows
  • Applications installées — nombre total détecté toutes sources
  • Processus actifs — nombre de processus en cours d'exécution

RAM Cleaner

Module de surveillance et nettoyage de la mémoire vive avec 18 opérations P/Invoke natives Windows.

Indicateurs surveillés

  • RAM physique — utilisée / totale
  • Mémoire virtuelle (Commit Charge) — utilisation RAM + pagefile
  • Taille du Pagefile — taille physique sur disque

Types de nettoyage (18 opérations P/Invoke)

  • Vider les Working Sets de tous les processus
  • Purger le Modified File Cache
  • Libérer la Standby List
  • Libérer la Priority 0 Standby List
  • Combiner les pages mémoire (Memory Combine)
  • Forcer le Garbage Collector .NET
  • Vider le cache système (System Cache)
  • Compresser la mémoire
  • Libérer la mémoire des processus inactifs
  • Purger le File Cache
  • Réduire le Working Set du processus courant
  • Appeler NtSetSystemInformation (MemoryPurge)
  • Forcer le vidage du Pool Non Paginé
  • Libérer les handles orphelins
  • Vider les Large System Cache
  • Forcer SetProcessWorkingSetSize sur tous les processus
  • Purger le session pool
  • Nettoyage complet combiné (all-in-one)

Mode automatique

Configurez un intervalle de nettoyage (ex : toutes les 30 minutes) ou un seuil d'utilisation (ex : dès que la RAM dépasse 85%).

Action IA : open_ramcleaner

Coclico AI

Assistant conversationnel intégré, alimenté par un modèle GGUF local via LLamaSharp v0.26. Aucune donnée ne quitte votre machine.

Caractéristiques techniques

  • Moteur : LLamaSharp 0.26 — inférence locale GGUF (CPU par défaut, Vulkan GPU optionnel)
  • Modèle fourni : IA-support-chat.gguf (~2.3 GB) — téléchargeable depuis l'interface
  • Context size : 2048 tokens
  • Réponses max : 512 tokens par message
  • RAG : top-K=2, 350 chars max — enrichit les réponses avec la doc Coclico
  • Mémoire : 4 échanges conservés, reset automatique après 6 tours
  • Nettoyage RAM : libère la mémoire après chaque réponse générée
  • GPU : tente Vulkan (8 couches GPU) puis retombe sur CPU automatiquement
Configuration recommandée : 16 Go de RAM minimum, processeur 4+ cœurs. 32 Go recommandé pour des performances optimales. Le modèle se télécharge (~2.3 GB) directement depuis le panneau Coclico AI.

Installation du modèle

Depuis l'interface Coclico AI (bouton en bas de la sidebar), cliquez sur "Télécharger le modèle IA". Le modèle se télécharge automatiquement (~2.3 GB) depuis Hugging Face et se place dans le bon dossier.

Alternativement, placez manuellement un fichier .gguf compatible dans :

Coclico\resource\model\IA-support-chat.gguf
Vie privée totale : Le modèle tourne entièrement en local. Aucune donnée n'est envoyée sur internet. Aucun compte requis.

Raccourcis Clavier Globaux

Configurez des raccourcis clavier globaux dans Paramètres → Raccourcis. Ils fonctionnent même lorsque Coclico est en arrière-plan ou réduit dans le tray.

Formats supportés

  • Ctrl + Touche
  • Ctrl + Alt + Touche

Chaque raccourci déclenche une Flow Chain spécifique identifiée par son nom. Plusieurs raccourcis peuvent être configurés simultanément.

Les raccourcis globaux sont gérés par KeyboardShortcutsService via les API Win32 RegisterHotKey / UnregisterHotKey.

Icône Tray

Coclico peut se réduire dans la zone de notification Windows (system tray) pour rester accessible sans encombrer la barre des tâches.

  • Clic gauche : réouvrir la fenêtre principale
  • Clic droit : menu contextuel (Ouvrir, Quitter)

Activez dans Paramètres → Réduire dans le tray. Le comportement à la fermeture (X) peut être configuré pour réduire dans le tray plutôt que quitter.

Profil Utilisateur

Personnalisez votre profil Coclico pour une expérience plus personnelle :

  • Avatar : importez une image (PNG, JPG) et recadrez-la avec l'outil intégré (recadrage carré)
  • Nom d'utilisateur : affiché dans la sidebar et l'écran d'accueil

L'avatar est sauvegardé dans %AppData%\Coclico\avatar.png

Architecture MVVM

Coclico utilise le pattern MVVM (Model-View-ViewModel) avec WPF .NET 10 et WPF-UI 4.2.

Coclico/
├── Views/          # Vues XAML + code-behind minimal (10 vues)
├── ViewModels/     # ViewModels (logique de présentation, 6 VMs)
├── Models/         # FlowChain.cs, NodeType (28 nœuds), LaunchMode
├── Services/       # 22 services découplés (ServiceContainer)
├── Converters/     # Convertisseurs WPF (6 convertisseurs)
└── Resources/
    ├── Lang/       # 10 fichiers XAML de traduction (fr, en, de, es, it, ja, ko, pt, ru, zh)
    ├── icone/      # Icône application (plage.ico / plage.png)
    ├── docs/       # Données RAG pour l'IA (JSON par langue)
    └── software_list.json  # Catalogue Winget curé

Couches applicatives

  • Views : XAML pur + minimal code-behind (events UI uniquement)
  • ViewModels : logique de présentation, binding, ICommand, INotifyPropertyChanged
  • Services : logique métier découplée, injectée via ServiceContainer maison
  • Models : données sérialisables (FlowChain, FlowItem, NodeType, AppSettings)

FAQ — Questions fréquentes

L'application ne démarre pas, que faire ?

Vérifiez que .NET 10 Desktop Runtime est installé. Téléchargez-le sur dotnet.microsoft.com. Vérifiez également les journaux dans %AppData%\Coclico\logs\ pour identifier l'erreur exacte.

Coclico nécessite-t-il des droits administrateur ?

Pour la plupart des fonctionnalités, non. Les droits administrateur sont recommandés pour : le RAM Cleaner (certaines opérations P/Invoke), le Nettoyage Système (fichiers système), et les Flow Chains exécutant des commandes système. L'app.manifest demande une élévation à la première utilisation de ces fonctions.

L'IA est très lente, comment accélérer ?

Coclico AI tourne entièrement en local (CPU). Assurez-vous d'avoir au moins 16 Go de RAM et un processeur récent (4 cœurs min.). Fermez les autres applications gourmandes avant d'activer l'IA. Si votre GPU supporte Vulkan, Coclico tentera automatiquement l'accélération GPU.

Comment changer la langue ?

Allez dans Paramètres (icône en bas de la sidebar) → section Langue → choisissez parmi les 10 langues disponibles. Le changement est immédiat, sans redémarrage.

Mes Flow Chains sont-elles sauvegardées automatiquement ?

Oui. Chaque modification est persistée automatiquement dans %AppData%\Coclico\flow_chains.json. Vous pouvez sauvegarder ce fichier pour migrer vos chaînes vers un autre PC.

Comment ajouter une application manuellement dans la bibliothèque ?

Allez dans Applications → cliquez sur le bouton + ou Ajouter → sélectionnez votre fichier .exe → assignez un nom et une catégorie → validez. L'application apparaît dans la bibliothèque et est persistée dans manual_apps.json.

Le RAM Cleaner peut-il endommager le système ?

Non. Toutes les opérations sont 100% réversibles — elles libèrent uniquement de la mémoire mise en cache que Windows peut recréer à la demande. Aucune donnée n'est supprimée. Les processus système critiques ne sont jamais affectés.

Comment désinstaller Coclico complètement ?

Passez par Paramètres Windows → Applications → cherchez "Coclico" → Désinstaller. Pour supprimer également vos données personnelles, supprimez manuellement le dossier %AppData%\Coclico\

Coclico collecte-t-il des données ?

Non. Coclico est 100% local. Aucune télémétrie, aucun compte requis, aucune connexion internet (sauf pour l'installeur Winget et le téléchargement optionnel du modèle IA). Toutes les données restent sur votre machine.

Comment créer un raccourci global pour une Flow Chain ?

Allez dans Paramètres → Raccourcis → cliquez sur Ajouter un raccourci → choisissez la Flow Chain cible → appuyez sur la combinaison de touches souhaitée (Ctrl+Touche ou Ctrl+Alt+Touche) → validez. Le raccourci fonctionne même quand Coclico est réduit dans le tray.
